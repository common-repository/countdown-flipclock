== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. A new menu item wil appear, named 'CountDown FlipClock'. Add your own clock and copy the shortcode shown below the publish button or only the ClockID.
5. Add the shortcode in any post, page or editor manually or use the button provided for the same. Then paste the ClockID within the quotes.
6. You're Done!!!
